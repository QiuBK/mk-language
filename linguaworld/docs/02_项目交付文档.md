# 多语种学习在线教育平台 - 项目交付文档

## 一、项目概述

### 1.1 项目背景

随着全球化进程加速，多语种学习需求日益增长。传统的线下语言培训存在成本高、时间不灵活、缺乏即时反馈等问题。本项目旨在打造一款功能完善的在线多语种学习平台，为用户提供便捷、高效、沉浸式的语言学习体验。

### 1.2 项目定位

LinguaWorld 是一款支持英语、日语、韩语等多语种的在线学习平台，通过互动式学习、个性化推荐和成就激励系统，帮助用户高效掌握目标语言。

### 1.3 核心目标

1. 提供多语种一站式学习服务
2. 实现个性化学习路径
3. 提供实时学习反馈
4. 支持学习数据化管理

### 1.4 开发周期与团队

- **开发周期**: 8周
- **前端开发**: 3人
- **后端开发**: 2人
- **UI设计**: 1人
- **产品经理**: 1人

### 4.2 核心表结构

#### 用户表 (tb_user)
```sql
CREATE TABLE tb_user (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) NOT NULL UNIQUE,
    password VARCHAR(255) NOT NULL,
    email VARCHAR(100) NOT NULL UNIQUE,
    nickname VARCHAR(50),
    avatar VARCHAR(255) DEFAULT '/assets/default-avatar.png',
    role TINYINT DEFAULT 0 COMMENT '0-用户,1-教师,2-管理员',
    level INT DEFAULT 1,
    exp INT DEFAULT 0,
    streak INT DEFAULT 0 COMMENT '连续学习天数',
    total_words INT DEFAULT 0,
    total_minutes INT DEFAULT 0,
    status TINYINT DEFAULT 1,
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

## 五、部署文档

### 5.1 环境要求

#### 开发环境
- Node.js >= 18
- JDK 17
- MySQL 8.0
- Redis 7.x
- Maven 3.8+

#### 生产环境
- CPU: 2核+
- 内存: 4GB+
- 磁盘: 50GB+
- 系统: Ubuntu 20.04 / CentOS 7+

### 5.2 部署步骤

#### 1. 数据库部署
```bash
# 安装MySQL
sudo apt update
sudo apt install mysql-server

# 创建数据库
mysql -u root -p
CREATE DATABASE linguaworld CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
EXIT;

# 导入数据
mysql -u root -p linguaworld < database/linguaworld.sql
```

#### 2. Redis部署
```bash
# 安装Redis
sudo apt install redis-server

# 启动Redis
sudo systemctl start redis
sudo systemctl enable redis
```

#### 3. 后端部署
```bash
# 进入后端目录
cd backend

# 编译打包
mvn clean package -DskipTests

# 运行
java -jar target/linguaworld-backend-1.0.0.jar

# 或使用nohup后台运行
nohup java -jar target/linguaworld-backend-1.0.0.jar > app.log 2>&1 &
```

#### 4. 前端部署
```bash
# 进入前端目录
cd frontend

# 安装依赖
npm install

# 开发环境运行
npm run dev

# 生产环境打包
npm run build

# 将dist目录部署到Nginx
sudo cp -r dist/* /var/www/linguaworld/
```

#### 5. Nginx配置
```nginx
server {
    listen 80;
    server_name your-domain.com;

    root /var/www/linguaworld;
    index index.html;

    # 前端静态资源
    location / {
        try_files $uri $uri/ /index.html;
    }

    # API代理
    location /api {
        proxy_pass http://localhost:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    # 静态资源
    location /uploads {
        alias /tmp/linguaworld/uploads;
        expires 7d;
    }
}
```

### 5.3 常见问题

#### Q1: 端口被占用
```bash
# 查找占用端口的进程
lsof -i:8080
# 杀死进程
kill -9 <PID>
```

#### Q2: 数据库连接失败
检查 `application.yml` 中的数据库配置是否正确，确认MySQL服务是否启动。

#### Q3: 录音权限问题
确保浏览器麦克风权限已开启，前端部署使用HTTPS。

## 八、维护文档

```bash
# 查看后端日志
tail -f logs/application.log

```

2. **数据备份**
```bash

mysqldump -u root -p linguaworld > backup_$(date +%Y%m%d).sql

# 备份上传文件
tar -czf uploads_backup_$(date +%Y%m%d).tar.gz /tmp/linguaworld/uploads
```

3. **服务监控**
```bash
# 检查服务状态
systemctl status nginx
systemctl status mysql
systemctl status redis

# 重启服务
systemctl restart nginx
systemctl restart java
```

### 8.2 故障排查

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 前端无法加载 | Nginx配置错误 | 检查nginx配置 |
