## 项目概述

LinguaWorld 是一款功能完善的在线多语种学习平台，支持 **英语、日语、韩语** 三大主流语言。

### 核心特性

- 🎓 多语种课程学习 - 分级课程体系
- 📝 智能单词背诵 - 艾宾浩斯遗忘曲线复习
- 🎤 口语 AI 评分 - 实时反馈与错误分析
- 📊 学习数据追踪 - 进度可视化与统计
- 🏆 成就激励系统 - 徽章与等级系统
- 👥 社区交流 - 学习心得分享

### 技术栈

**前端：**
- Vue 3 + TypeScript + Vite
- Element Plus + Tailwind CSS
- Pinia 状态管理
- Axios HTTP 请求

**后端：**
- Node.js + Express（简化版）
- SQLite 数据库
- JWT 认证
